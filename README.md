# CE-STAND Visual N-Back Test

A web-based **Visual N-Back cognitive task** developed for the **CE-STAND study** at the **University of Minnesota – Department of Neurosurgery**.  
The application measures working memory and attention using a visual stimulus matching task.

## Overview
Participants observe a square appearing in different positions on a grid.  
The goal is to determine whether the current square appears in the **same position as N steps earlier** in the sequence.

Supported levels:
- **1-Back**
- **2-Back**
- **3-Back**

The task evaluates cognitive performance using:
- **Score** (correct responses)
- **Accuracy**
- **Wrong responses**
- **Reaction time**

## Controls
- **S key** → Press when the current square matches the position **N steps ago**
- **Start button** → Begin the session

## Session Structure
- 20 observations per session
- 1 point awarded for each correct response
- Incorrect or missed responses receive 0 points

## Technology
- HTML
- CSS (Bootstrap)
- JavaScript

## Usage
Open `index.html` in a browser or deploy via **GitHub Pages**.

## Study Context
This tool is used within the **CE-STAND cognitive assessment framework** to evaluate working memory, sustained attention, and processing speed across sessions.
