How to Play

When you press the Start button, the application will begin presenting a square that briefly appears in one of the positions in the grid. Your task is to remember the location of each square as it appears.

The game uses an N-back working memory task. The level (1-back, 2-back, or 3-back) determines how many steps back you must remember.

  • 1-Back: Compare the current square with the square shown 1 step earlier.
  • 2-Back: Compare the current square with the square shown 2 steps earlier.
  • 3-Back: Compare the current square with the square shown 3 steps earlier.

If the square appears in the same location as N steps ago, press the S key.

Each session contains 20 observations (trials). On each trial:

  1. A square briefly appears in one of the grid locations.
  2. You decide whether its location matches the location shown N steps earlier.
  3. If it matches, press S. If it does not match, do nothing.

After the session ends, the system will display several performance metrics.

Performance Metrics

  • Score: Number of correct responses. Each correct response earns 1 point. Maximum score per session is 20.
  • Correct Responses: Number of times the participant correctly identified a matching location.
  • Wrong Responses: Number of times the participant pressed S when there was no matching location (false positive).
  • Missed Targets: Number of times a match occurred but the participant did not press S.
  • Accuracy: Percentage of correct responses relative to the total number of trials.
  • Reaction Time: The average time (in milliseconds) between when the square appears and when the participant presses S for correct responses.

This task measures working memory, attention, and cognitive processing speed. It is commonly used in cognitive neuroscience and clinical research to assess executive function and memory performance.

About

The Visual N-Back task is used in the CE-STAND study at the University of Minnesota Department of Neurosurgery as a computerized cognitive assessment of working memory and attention. During the task, a square briefly appears in different positions within a grid on the screen. Participants must remember the position of the squares as they appear over time and determine whether the current square appears in the same location as one presented a specified number of steps earlier (the N-Back level). Depending on the selected level (1-Back, 2-Back, or 3-Back), participants compare the current stimulus with the square that appeared one, two, or three steps earlier. When a match occurs, participants respond using the keyboard. This task places demands on working memory, sustained attention, and cognitive processing. Within CE-STAND, it is used as a standardized tool to measure cognitive performance and track changes in attention and memory across testing sessions.
